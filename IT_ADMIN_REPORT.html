1. Executive Overview

File List Generator is a simple Windows utility that adds a "Generate File List" option to the right-click menu in Windows Explorer. When users right-click a folder and select this option, a window opens showing all files in that folder with search and sorting capabilities.

Primary Function: Display folder contents in a searchable, sortable list that can be copied to Excel or other applications.

3.2 Registry Modifications

The installer adds entries to enable the right-click menu option. All entries are in HKEY_CURRENT_USER (user-specific, not system-wide):

Location: HKEY_CURRENT_USER\Software\Classes\Directory\shell\FileListGenerator Purpose: Adds "Generate File List" to folder right-click menu Location: HKEY_CURRENT_USER\Software\Classes\Directory\Background\shell\FileListGenerator Purpose: Adds option when right-clicking inside a folder Location: HKEY_CURRENT_USER\Software\Classes\Drive\shell\FileListGenerator Purpose: Adds option for drive letters (C:\, D:\, etc.)

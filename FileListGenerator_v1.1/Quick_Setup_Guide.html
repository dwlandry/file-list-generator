üìä How to Use

Opening the Program

After installation, you have two ways to scan folders:

Method 1: Right-Click Menu

  1. Navigate to any folder in Windows Explorer
  2. Right-click the folder (or inside it)
  3. Select "Generate File List"

Method 2: Drag & Drop

  1. Open File List Generator from any folder
  2. Drag a different folder from Windows Explorer
  3. Drop it into the File List Generator window
  4. The new folder's contents appear instantly!
